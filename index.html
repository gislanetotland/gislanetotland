Learn C#

  • Codecademy

I dette kurset vil du bli utsatt for grunnleggende programmeringskonsepter ved hjelp av C# og begynne å skrive programmer med en gang. Du vil bygge flere prosjekter for å hjelpe deg med å øve og teste kunnskapen din ved hjelp av quizer.

Python for Data Science

  • Pågår
  • DataCamp

Dette kurset fokuserer på Python spesielt for datavitenskap. I introduksjon til Python-kurset vårt vil du lære om kraftige måter å lagre og manipulere data på, og nyttige datavitenskapelige verktøy for å begynne å utføre dine egne analyser.

Responsive Webdesign

  • Pågår
  • freeCodeCamp

Først skal du bygge en kattefotoapp for å lære det grunnleggende om HTML og CSS. Senere vil du lære moderne teknikker som CSS-variabler ved å bygge en pingvin, og beste praksis for tilgjengelighet ved å bygge et nettskjema.

Til slutt vil du lære hvordan du lager nettsider som reagerer på forskjellige skjermstørrelser ved å bygge et Twitter-kort med Flexbox, og et komplekst bloggoppsett med CSS Grid.
